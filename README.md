### How to Run?

---

1. Clone this Project
2. type `code .`
3. On your Code editor, open terminal and then type `npm install`
4. Run the project by typing `npm run dev`
5. Happy Testing

#### Link deploy : https://netflix-clone-david.vercel.app/
